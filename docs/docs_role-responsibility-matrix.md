# Role & Responsibility Matrix (RACI) — Template

Use this RACI template to clarify who is Responsible / Accountable / Consulted / Informed for major project activities.

| Activity / Deliverable | Project Manager (PM) | Product Manager (PdM) | Developers | QA Lead | Release Engineer | Stakeholder Liaison | Tech Writer |
|---|---:|---:|---:|---:|---:|---:|---:|
| Define success metrics | A | R | C | C | I | C | I |
| Create Project One-pager | R | A | I | I | I | C | I |
| Backlog prioritization | I | A | C | C | I | C | I |
| Development and code changes | I | C | R | C | I | I | C |
| Test strategy & acceptance | I | C | C | R | I | I | C |
| Release planning | R | C | C | C | A | C | I |
| Release notes & docs | I | C | I | I | C | I | R |
| Incident response (post-release) | R | I | R | C | A | I | I |

Legend:
- R = Responsible (does the work)
- A = Accountable (final decision/ownership)
- C = Consulted (two-way input)
- I = Informed (one-way communication)

Instructions:
- Copy this template into your project kickoff artifacts and fill in per-activity ownership changes as needed.
- Add owner names and contact emails in your Project One-pager.
